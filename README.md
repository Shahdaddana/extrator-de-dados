# Extrator de dados

### Para receber o código pix de pagamento da Energisa, digite a matrícula do seu imóvel

* GET /energisa/receberCodigoPix?matricula=X

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
